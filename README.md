## Getting Started

To run this version locally:

1. Clone the repository:
```bash
git clone https://github.com/Teodor29/portfolio-v1.git

```
2. Navigate to the project folder:
```
cd portfolio-v1
```
3. Install dependencies:
```
npm install
```
4. Start the development server:
```
npm run dev
```
